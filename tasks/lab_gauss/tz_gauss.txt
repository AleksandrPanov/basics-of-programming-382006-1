Даны два файла gauss.cpp и check_utilities.h. Необходимо:
1.1 Создать пустой проект VS, скопировать в папку с проектом файлы, потом добавить их в проект.
(Для gauss.cpp: Исходные файлы -> ПКМ -> Существующий элемент... -> выбрать нужный файл)
(Для check_utilities.h: Файлы заголовков -> ПКМ -> Существующий элемент... -> выбрать нужный файл)
1.2 Скомпилировать (проект компилируется у вас должен компилироваться, если нет, то вы что то неправильно сделали).

2.1 Понять, что делают функции в файле check_utilities. Реализация функции умножения матрицы на вектор может быть на экзамене или в тесте.
2.2 Обратите внимание, функция generate не выделяет память для матрицы A и вектора b. В си принято, что функции выделяют память только для локальных переменных.
(так делает функция check)
2.3 check_utilities.h - отдельный заголовочный файл, чтобы его использовать, его нужно "включить" в файл с исходным кодом (в cpp файл). Это делается директивой #include "ИмяФайла.h"

3.1 Необходимо реализовать функцию void gauss_forward(double** A, double* b, const int n)
Функция должна преобразовать матрицу A к верхнетреугольной (элементы на главной диагонали не равны нулю).
3.2 Необходимо реализовать функцию void gauss_backward(double** A, double* b, double *x, const int n)
Функция должна найти решение системы и записать в x

4.1 Функция check должна вернуть число меньшее 0.000001
